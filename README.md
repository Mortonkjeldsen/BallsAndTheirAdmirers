# C template

For VSCode-based editors and Clang. Requires CodeLLDB extension. clangd extension is recommended.

An assignment for the DTU course Basic C Programming

The program creates a 400x400 window, initializes 100 random balls that follow other random balls, and then draws them all. The result is a constantly shifting, colorful pattern of moving circles.
